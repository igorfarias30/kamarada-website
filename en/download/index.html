Below we present some options to download the openSUSE Linux distribution, recommended by the Linux Kamarada Project.

openSUSE Leap 42.1

  • 64-bit IBM PC's
  • Stable and mature software
  • Released on November 2015
  • Will receive updates until November 2018

Recommended for most users.

If that is the system you want, click on the following link to start downloading.

openSUSE Leap 42.1

Release Notes

Once you click the download link, you will get the LiveDVD with KDE desktop built by the Linux Kamarada Project.

If you want more options, check the openSUSE Leap 42.1 official download page.

openSUSE 13.2

  • 32-bit IBM PC's
  • Stable and mature software
  • Released on November 2014
  • Will receive updates until April 2017

For legacy or low memory computers.

If that is the system you want, click on the following link to start downloading.

openSUSE 13.2

Release Notes

Once you click the download link, you will get the official openSUSE 13.2 LiveDVD with KDE desktop.

If you want more options, check the openSUSE 13.2 official download page.

openSUSE Tumbleweed

  • 32 or 64-bit IBM PC's
  • Stable and brand new software
  • Continuously receives updates (rolling release)

For those who want to use bleeding edge software.

If that is the system you want, click on one of the following links, according to your computer's architecture, to start downloading.

32-bit Tumbleweed

64-bit Tumbleweed

Once you click one of the links above, you will get the official openSUSE Tumbleweed LiveDVD with KDE desktop.

If you want more options, check the openSUSE Tumbleweed official download page.

You will download Linux in the form of an ISO image: a single file that you will later burn on a DVD, install onto a USB stick or memory card, or use with a virtual machine.

We recommend checking out the Release Notes before installation.
